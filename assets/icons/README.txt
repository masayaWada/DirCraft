このディレクトリのアイコン (SVG および PNG) は、Wikimedia OOUI
(https://github.com/wikimedia/oojs-ui) の
src/themes/wikimediaui/images/icons/ 配下にある SVG を元にしている。

原典と本プロジェクトでの呼称の対応:
  add.svg        <- add.svg
  cancel.svg     <- cancel.svg
  check.svg      <- check.svg
  clear.svg      <- clear.svg
  close.svg      <- close.svg
  copy.svg       <- copy-ltr.svg
  file.svg       <- tray.svg
  folder.svg     <- folderPlaceholder-ltr.svg
  reload.svg     <- reload.svg
  settings.svg   <- settings.svg

PNG は rsvg-convert で 20x20 px にラスタライズしたもの。
